# Two-Day Delivery Plan

## Day 1

- Build backend solution structure.
- Implement domain model and workflow transitions.
- Implement EF Core persistence and SQL Server setup.
- Implement request workflow API endpoints.
- Add unit tests for workflow.
- Draft README, architecture, database schema, and runbook.

## Day 2

- Implement Data Integration Repair Center service and stored procedures.
- Implement Angular dashboard and import repair UI.
- Add import validation tests.
- Add GitHub Actions CI.
- Complete incident case study and test plan.
- Run build/test verification in a .NET 8 environment.

## Scope control

Do not add real ServiceNow integration, real SSO, cloud deployment, or microservices in this two-day build. Those are follow-on issues.
